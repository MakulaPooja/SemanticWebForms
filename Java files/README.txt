> The folder named 'Semantic_forms' can be imported as an eclipse project.
> The input file used in Semantic_forms/src/ontology/Main.java is 'pizza.owl' which is given in 'Java\ files/' folder.
> The API 'apache-jena-3.0.0' folder under 'Java\ files/' is the external api which needs to be added as an external library for the project.

---------------------------------------------------------------------------------------------------------------------------------------------------

	-> Instructions to add jena api:	
		- in the package explorer in eclipse workspace, select "Build Path -> Configure build path -> Libraries -> Add Library -> User 			  Library"
		- then hit "next" and follow the below steps:
			- select "User Libraries -> New"
			- give the name to the new library and,
			- select "Add external JARs" and, 
			- select all the .jar files from "Java\ files/apache-jena-3.0.0/lib/" and hit OK

----------------------------------------------------------------------------------------------------------------------------------------------------
